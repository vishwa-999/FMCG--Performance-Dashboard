# ⚙️ Automated IFRS Cashflow Engine (Indirect Method)
### Standardized Financial Automation // IAS 7 & IFRS Compliant

This repository contains a **Python-based financial logic engine** designed to automate the production of Cash Flow Statements from raw P&L and Balance Sheet data. Built specifically for **SME Finance Transformation**, it eliminates manual spreadsheet errors and ensures compliance with **IFRS (IAS 7)** reporting standards.

## 💎 Core Value Proposition
* **Compliance First:** Follows ACCA/FCA guidelines for the Indirect Method, ensuring audit-ready outputs.
* **Working Capital Intelligence:** Automatically calculates cash impacts of Receivables, Payables, and Inventory movements using the Asset/Liability inverse logic.
* **Separation of Tax/Interest:** Strictly adheres to IAS 7 requirements for separate disclosure of financing and tax cash flows.

## 🚀 Impact for Growth Agencies
In fast-paced agencies like **Croci Collective**, the "Reporting Lag" usually comes from manual working capital reconciliations. This script:
1. **Reduces Prep Time:** From hours to milliseconds.
2. **Improves Accuracy:** Standardizes the logic for Working Capital "Inflow vs Outflow."
3. **Strategic Insight:** Allows for instant "What-If" scenarios on cash runways.
